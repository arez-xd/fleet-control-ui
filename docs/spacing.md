# Spacing

Spacing is based on a fixed scale:

0 / 4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 / 48

---

## Rules

- All spacing must use tokens
- Arbitrary values are not allowed
- Larger spacing implies stronger separation

---

## Typical usage

- 8–12: inline spacing
- 16–24: component padding
- 32–48: layout sections
